# YouTubeMusicAPI

A Python package for searching songs on [YouTube Music](https://music.youtube.com/).

```
pip install YouTubeMusicAPI==2.6.0
```

## Example

```python
from YouTubeMusicAPI import Search

result = Search("Alan walker faded")

print(result)
```
